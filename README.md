&nbsp;&nbsp;&nbsp;&nbsp;bloxtool-go record:cname update \<alias\> \<cname\> \<view\>  
&nbsp;&nbsp;&nbsp;&nbsp;bloxtool-go record:cname delete \<alias\> \<view\>
